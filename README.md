#  Credit Risk Prediction using Machine Learning

##  Overview

This project analyzes credit-related customer data and builds machine learning models to predict credit risk / approval outcomes. It follows a complete ML workflow including data preprocessing, feature engineering, scaling, model training, and performance comparison.
The goal is to identify the most effective algorithm for credit prediction based on evaluation metrics.

##  Objective

* Analyze credit dataset features
* Clean and preprocess raw data
* Engineer and transform features
* Train multiple ML models
* Compare model performance
* Select the best performing model
